About Joe

I recently got back into coding specifically because of the idea of web development; It is a medium with unlimited possiblities due to the power of code and design. I'm excited to see what I can offer to the table!

Focuses


Clean Code, documentation, and following best practice

I want to make code that anyone can read. This means to me following the principles of "Clean Code" by Robert C. Martin, making any complex methods or sections of code well commented, and finally constantly learning and refactoring my code.

Fast and easy to use.

I want my code to be fast and reliable. I am a firm believer that products and services should just work for the user without any head scratching.

Layout

Layout is something I'm focusing on right now. I want to know all of the ways to position elements whether it be bootstrap columns, CSS grid, or flexbox as well as making sure I know the best use cases for each.

Planning and workflow

Everytime I set out to create a project I want to make sure I have a strong plan and a good road map. To me this is basically just writing the directions to the project and breaking it up into managable sections. Then all thats left is following the instructions.
